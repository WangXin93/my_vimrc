This is my configuration for vim.
## How to use
Input this in you terminal:
```
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim \
git clone https://github.com/WangXin93/my_vimrc.git \
cp my_vimrc/.vimrc ~ \
cp -R my_vimrc/.vim ~ \
vim # open vim 
```
Then input in vim:
```
:PluginInstall
```
Done.
